ยินดีต้อนรับเข้าสู่เว็บไซต์แนะนำสถานที่ท่องเที่ยวจังหวัดพิษณุโลก   13 ตุลาคม วันนวมินทรมหาราช น้อมลำรึกในพระมหากรุณาธิคุณอย่างหาที่สุดมิได้

- จุดเช็คอินสายมู จังหวัด พิษณุโลก -

- แนะนำสถานที่เหล่าสายมูเตลู -

       เมืองสองแคว วัดพิษณุโลก หลายคนคงได้มา จังหวัดพิษณุโลกกันบ้างแล้ว เหล่าเพื่อนๆสายมูคงหาวัดทำบุญกันอยู่ใช่มั้ยค่ะ วัดในพิษณุโลก มีทั้งวัดที่เก่าแก่ วัดดังเดิม วัดสวยๆ วัดที่พิษณุโลก และยังมีแหล่งให้ได้ศึกษาธรรมะ พิษณุโลกวัด สำหรับใครที่เดินทางท่องเที่ยว ที่เที่ยวสุโขทัย ที่เที่ยวกำแพงเพชร ที่เที่ยวพิจิตร กำลังมองหา วัดในพิษณุโลก วัดที่พิษณุโลก สามารถแวะเข้ามาทำบุญ วัดพิษณุโลก กันได้อย่างสะดวกเลยค่ะ

- วัดอรัญญิก -

     วัดอรัญญิก เป็นวัดโบราณ พระเจดีย์ทรงลังกาที่ตั้งอยู่ภายในวัด สามารถสันนิษฐานได้ว่าพระเจดีย์นั้นสร้างขึ้นในสมัยสุโขทัย ทรงลังกาเป็นสถาปัตยกรรมที่ใช้ในการประดิษฐ์พระบรมสารีริกธาตุ หรือพระเจดีย์เพื่อเป็นที่บูชาและเคารพบูชาในศาสนาพุทธ วัดศักดิ์สิทธิ์ พิษณุโลก

Google Map : https://goo.gl/maps/8s4er2EVZmZsn5rP7
พิกัด : ตำบลในเมือง อำเภอเมืองพิษณุโลก จังหวัดพิษณุโลก

- วัดจุฬามณี -

     วัดจุฬามณี ภายในวัดได้มีโบราณที่สำคัญ คือ พระมณฑปพระพุทธบาทจำลองและศิลาจารึกค่ะ โดยสมเด็จพระนารายณ์มหาราชได้ทรงโปรดให้สร้างขึ้นมา และยังมี พระปรางค์แบบขอมขนาดเล็กที่ก่อด้วยศิลาแดง

Google Map : https://goo.gl/maps/nj5SLWDfKCF2RemN8
พิกัด : 50/3 หมู่ 2 ริมแม่น้ำน่าน ตำบลท่าทอง อำเภอเมืองพิษณุโลก

- ศาลพระนเรศวร -

     ศาลสมเด็จพระนเรศวรมหาราช ที่ตั้งอยู่ในบริเวณของโรงเรียนพิษณุโลกพิทยาคม ซึ่งศาลแห่งนี้เป็นศาลาทรงไทยโบราณตรีมุขพระรูปสมเด็จพระนเรศวรมหาราชที่มีขนาดเท่ากับองค์จริง นั่งประทับอยู่ด้านใน โดยมีพระหัตถ์ทรงพระสุวรรณภิงคารหลั่งน้ำในอิริยาบถประกาศอิสรภาพที่เมืองแครง ทุกวันที่ 25 มกราคม ของทุกปี จะมีการจัดงานนมัสการพระบรมรูป

Google Map : https://goo.gl/maps/8UKXW2C5n34tBokY8
พิกัด : ตำบลบ้านคลอง อำเภอเมืองพิษณุโลก จังหวัดพิษณุโลก

- วัดนางพญา -

     วัดนางพญา วัดแห่งนี้จะโด่งดังในเรื่องของพระพิมนางพญา ซึ่งมักจะมีนักท่องเที่ยวแวะเวียนกันมากราบไหว้เพื่อขอพรกันเป็นจำนวนมาก ภายในอุโบสถด้านในได้มีลักษณะเป็นเรือนแก้ว ถูกประดับด้วยลวดลายบนฝาผนังต่างๆ มากมาย และยังมีความเชื่อกันว่าพระพิมนางพญานั้นถูกสร้างขึ้นโดยพระสิสุทธิกษัตริย์

Google Map : https://goo.gl/maps/WD8wmRm2ZYLnpNf16
พิกัด : ถนนจ่าการบุญ อำเภอเมืองพิษณุโลก จังหวัดพิษณุโลก

- วัดราชบูรณะ -

     วัดราชบูรณะ เป็นที่ประดิษฐานขององค์หลวงพ่อทองคำ ที่เป็นหนึ่งในพระพุทธรูปปั้นลงรักปิดทองมารวิชัย โดยภายในวัดยังมีโบราณสถานและโบราณวัตถุอีกมากมาย มีเจดีย์หลวงทรงลังกา ที่มีความสูงถึง 24 เมตร ถือได้ว่าเป็นเจดีย์ที่มีความสูงที่สุดในจังหวัดพิษณุโลก

Google Map : https://goo.gl/maps/za64rpiJCN4zpp3a6
พิกัด : ถนนมิตรภาพ ตำบลในเมือง อำเภอเมืองพิษณุโลก

- วัดจันทร์ตะวันตก -

     วัดจันทร์ตะวันตก มีการปรากฏในพระราชพงศาวดารฉบับพระราชหัตถเลขาหลายตอนด้วยกัน ซึ่งตรงกับในสมัยกรุงธนบุรีนั่นเองค่ะ วัดในจังหวัดพิษณุโลก มีกี่วัด ภายในวัดก็จะมี มหาวิหารสมเด็จองค์ปฐม และ พระพุทธรูปสมเด็จองค์ปฐม ปางมารวิชัย ทรงเครื่องมหาจักรพรรดิ อีกด้วยค่ะ

Google Map : https://goo.gl/maps/7dGmWmwrG5gwQCqu5
พิกัด : ตำบลวัดจันทร์ อำเภอเมืองพิษณุโลก จังหวัดพิษณุโลก

- วัดจันทร์ตะวันออก -

     วัดจันทร์ตะวันออก วัดตรงข้ามวัดจันทร์ตะวันตก สร้างขึ้นมาตั้งแต่ปี พ.ศ. 2276 เป็นจุดที่เชื่อกันอีกว่า พระยาจักรี หรือ ในหลวงรัชกาลที่ 1 เคยยกทัพและมาหยุดพักที่นี่ ภายในวัดนอกจากพระพุทธรูปต่างๆ แล้ว ก็ยังมี ศาลสมเด็จพระสุพรรณกัลยา วีรสตรีผู้เสียสละเพื่อประเทศชาติที่ชาวพิษณุโลก

Google Map : https://goo.gl/maps/Hzj2PzDTit8vQ6Rt8
พิกัด : ตำบลวัดจันทร์ อำเภอเมืองพิษณุโลก จังหวัดพิษณุโลก

- วัดพระพุทธชินราช -

     วัดพระพุทธชินราช หรือ วัดพระศรีรัตนมหาธาตุราชวรมหาวิหาร เป็นที่ประดิษฐานของ วัดพระพุทธชินราช พระพุทธรูปปางมารวิชัยหล่อด้วยทองสัมฤทธิ์ที่งดงามมากๆ มีความศักดิ์สิทธิ์ และเป็นที่พักพิงใจยึดเหนี่ยวจิตใจ ของชาวพิษณุ

Google Map : https://goo.gl/maps/2sLaFipxNHVSSL7i9
พิกัด : ตำบลในเมือง อำเภอเมือง จังหวัดพิษณุโลก

- วัดกกไม้แดง -

     วัดกกไม้แดง เป็นวัดที่มีพระพุทธรูปนอนองค์ใหญ่สีทองอร่ามตั้งอยู่ วัดเก่าแก่ พิษณุโลก ที่พักเนินมะปราง ถูกสร้างขึ้นมาตั้งแต่ในปี พ.ศ. 1580 แล้ว โดยตั้งชื่อตามชื่อหมู่บ้าน ที่มี บึงกกไม้แดง อยู่ทางทิศตะวันตกนั่นเอง นอกจากพระพุทธรูปนอนองค์นี้แล้ว ภายในวัดก็ยังมี เจ้าแม่กวนอิม ให้เราได้สักการะกันด้วยค่ะ

Google Map : https://goo.gl/maps/QDjCNNgrkyFjG27u6
พิกัด : ตำบลดินทอง อำเภอวังทอง จังหวัดพิษณุโลก
